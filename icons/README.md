# 图标文件说明

## 需要放入以下文件

- **icon16.png** - 16x16 像素（地址栏、扩展菜单显示）
- **icon48.png** - 48x48 像素（扩展管理页面、通知栏）  
- **icon128.png** - 128x128 像素（Chrome Web Store、大图标显示）

## 推荐来源

### 免费图标库
- 🔗 [Flaticon](https://www.flaticon.com/) - 搜索 "bookmark" 或 "bookmark ai"
- 🔗 [Icons8](https://icons8.com/) - 高质量图标
- 🔗 [Pixabay](https://pixabay.com/) - 免费高分辨率图片
- 🔗 [Unsplash](https://unsplash.com/) - 免费照片

### 设计工具
- 🎨 [Figma](https://www.figma.com/) - 在线设计
- 🎨 [Canva](https://www.canva.com/) - 快速设计

## 技巧

1. **下载高分辨率版本** - 建议至少 512x512 或更大，然后缩放
2. **PNG 格式必须** - Manifest 配置的是 PNG 格式
3. **透明背景更美观** - PNG 支持透明度，推荐用透明背景
4. **统一设计风格** - 3 个尺寸保持同一风格

## 将文件放入此目录

下载好图片后，重命名为上述文件名，放入此目录即可。
